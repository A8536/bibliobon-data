# Generated Data

This directory is for generated machine-readable data.

Planned files:

- `bibliobon.sqlite` - canonical export database for the Django site.
- `site_contract.json` - contract checked by the Django importer.
- `build_manifest.json` - summary of the latest build.
- `*.jsonl` - optional table-level exports for review and diagnostics.

Do not manually edit generated files. Put manual corrections in `data/manual/`.
